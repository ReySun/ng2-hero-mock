# HeroMock

This project was generated with [angular2中文网](https://angular.cn/) version 1.0.0-beta.28.3.

## 安装包依赖
防qiang，你懂的！
``` bash
npm install -g cnpm 
cnpm install
```

## 启动应用
默认启动[localhost:4200](http://localhost:4200/)
``` bash
npm start
```
如果端口被占用，可以使用ng serve命令换端口号,例如：
``` bash
ng serve --port=4000
```
